FINAL HAVS BIRTHDAY MEMORY ENGINE

LAYOUT
- Desktop/tablet: 3 cards per row.
- Mobile: 1 card per row.
- A clear scroll cue is shown beneath the introduction.
- Images and videos use object-fit: contain, so faces are not cropped.

MEDIA FILENAMES (CASE-SENSITIVE)
Upload front/reveal photos to images/ exactly as listed in images/UPLOAD_IMAGES_HERE.txt.
Upload videos to videos/ exactly as listed in videos/UPLOAD_VIDEOS_HERE.txt.

GITHUB UPDATE
1. Open your Havs-Birthday repository.
2. Add file > Upload files.
3. Drag EVERYTHING inside this folder onto GitHub.
4. Commit changes.
5. Then upload your actual JPEGs inside images/ and MP4s inside videos/.
6. Wait around one minute and hard-refresh the live page.

MUSIC
No music is playing by default. This avoids unwanted autoplay and works better on iPhone.
Optional instructions are inside audio/OPTIONAL_MUSIC.txt.

IMPORTANT
GitHub filenames are case-sensitive. Image1.jpeg is not the same as image1.jpeg.
Keep videos reasonably compressed for mobile loading. H.264 MP4 is the safest format for iPhone and Android.
